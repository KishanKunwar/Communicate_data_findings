Factors that impact the loan approval for an applicant¶

Dataset Overview¶

The Dataset that I choosed for this project of visulization is called "propserLoandata" . This data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.

For simplicity of the visualization I have looked at only few variables as the project objective itself says to focus on only few variables.
